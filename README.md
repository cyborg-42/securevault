## 📖 Overview

SecureVault is a desktop-based file encryption application designed to provide secure and user-friendly protection for digital files using modern cryptographic techniques.

The project combines **symmetric encryption (AES-256)**, **asymmetric encryption (RSA-2048)**, **password-based key derivation (PBKDF2)**, and **SHA-256 hashing** to ensure confidentiality, integrity, and secure key management.

Unlike conventional encryption tools that simply encrypt and decrypt files, SecureVault is designed with a modular architecture, allowing future enhancements such as cloud storage integration, digital signatures, and secure key sharing.

---

## 🎯 Objectives

- Secure sensitive files using AES-256 encryption.
- Protect encryption keys using RSA-2048.
- Generate secure encryption keys from user passwords using PBKDF2.
- Verify file integrity using SHA-256 hashing.
- Provide a clean and intuitive JavaFX desktop interface.
- Demonstrate secure software engineering practices through modular design.
